¿Qué son las etiquetas HTML?

Las etiquetas HTML son el camino que se va haciendo para desarrollar una Web.

Todo lo que escribimos o subimos irá dentro de las etiquetas, que le dirán a la Web qué es lo que vamos haciendo.

Por un lado tendremos etiquetas para estructurar, que le dirán a la Web si lo que estamos escribiendo en su interior es la cabecera de nuestra página, es el cuerpo, es el pie de página o es un elemento aparte.

También tendremos etiquetas para indicar si nuestra intención es escribir un párrafo, subir una imagen, crear una lista, o para que la Web sepa qué palabras de nuestro texto son clave y nuestra página aparezca en el buscador cuando alguien trate de buscar contenido a través de esas palabras.

Las etiquetas han sido diseñadas con la forma de un texto rodeado por corchetes angulares  <etiqueta>

Pueden ser:

Sencillas (One-sided tag)

Por ejemplo, para indicar que queremos realizar un salto de línea podría usarse la etiqueta <br>

Dobles (Two-sided tag)

Donde  <etiqueta>  es la etiqueta de apertura y  </etiqueta>  es la etiqueta de cierre, indicada por la diagonal.

Por ejemplo, para escribir un párrafo, primero indicamos con la etiqueta de apertura que el texto que va entre estas dos etiquetas es un párrafo, después escribimos nuestro párrafo y por último lo cerramos con la etiqueta de cierre.

Ejemplo de párrafo bien estructurado:

<p>Caminante, no hay camino, se hace camino al etiquetar</p>

Recuerda siempre cerrar todas las etiquetas dobles o tu página irá presentando errores.

Atributos para tus Etiquetas

Dentro de las etiquetas se pueden colocar una serie de Atributos

Los atributos sirven para especificar los detalles de las etiquetas que los contienen.

Pueden darle un estilo al contenido de la etiqueta, por ejemplo si usas el atributo style, o pueden llevar a una URL añadiendo a la etiqueta <a> el atributo href, así como indicar un título con title.

De esta manera hacemos que la web nos entienda.

Los atributos se escriben a la derecha de su etiqueta, dentro de los corchetes angulares que la contienen, separados de ésta únicamente por un espacio.

¿Y cómo especificamos los detalles de este atributo?

Una vez escrito el atributo dentro de la etiqueta, utilizaremos el símbolo "igual que" (=) y especificaremos sus detalles dentro de unas comillas.

Es mejor editar los estilos en CSS a través de los atributos id y class, pero eso lo veremos más adelante.

Etiquetas y atributos afines

HTML nos ha definido una serie de atributos globales, que se pueden utilizar en cualquier etiqueta, como es el caso de los atributos de estilo (le puedo asignar un estilo tanto a un párrafo como a todo el cuerpo de mi página, por ejemplo) pero también existen atributos asignados a cierto tipo de etiquetas, y etiquetas que obligatoriamente han de tener un atributo.

Por ejemplo, si quiero subir una imagen usaré una etiqueta simple <img>, pero necesariamente tendré que usar el atributo src para indicar el enlace de la imagen que quiero subir, ya sea una imagen local o una imagen de la web.
